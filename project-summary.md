## Overview of Project

World Bank Data Explorer is a client-side web application that pulls live economic data from the World Bank API and presents it in two formats: comparative line charts and summary cards. Users can select economies or regions, switch between chart and card views, and inspect indicators such as population, GDP, GDP per capita, and recent growth rates. The project combines asynchronous data fetching, DOM updates, and chart rendering in a single interactive interface.
