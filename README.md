# World Bank Data Explorer

A browser-based data explorer that visualizes World Bank economic indicators in charts and summary cards.

## Overview

World Bank Data Explorer is a front-end web project that lets users inspect country and regional data from the World Bank API. Users can compare multiple economies in chart view or switch to card view to review a single economy's latest population, GDP, GDP per capita, and recent growth metrics.

The interface uses asynchronous API requests to load live data, updates the page dynamically based on user input, and presents the results through both Chart.js line charts and DOM-generated information cards.

## Features

- Fetches live World Bank data based on the selected economy or region
- Displays population and GDP trends in comparative line charts
- Generates summary cards with population, GDP, GDP per capita, and growth rates
- Supports multiple view modes for comparison and quick inspection
- Expands the economy selector with additional World Bank entries on demand
- Shows in-page error messages when data is unavailable

## Built With

- HTML5
- CSS3
- JavaScript
- Fetch API
- Chart.js
- World Bank API

## Purpose

This project was built to practice client-side API integration, DOM manipulation, and data visualization in a single-page web interface.
